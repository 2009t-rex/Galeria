Co to modal?

Modal wyskakujące okno dialogowe, które przerywa interakcję użytkownika ze stroną, blokując dostęp do reszty treści do momentu zamknięcia okna.

Jak stworzyć modal?

Modal tworzymy za pomocą znacznika <dialog>.
Po stworzeniu element nie wyświetli się sam na stronie, posiada display: none.
Żeby, go otworzyć potrzeba guzika, który go pokaże. Poniżej pokarze kilka sposobów jak to zrobić.

Jak pokazać modal?

Sposób 1 - zmiana display

Wykorzystujemy JavaScript do zmiany właściwości display.

To jest modal otwarty poprzez zmianę display.
const modal = document.getElementById("modal");
const openBtn = document.getElementById("openModal");
const closeBtn = document.getElementById("closeModal");

openBtn.onclick = () => {
  modal.style.display = "block";
}

closeBtn.onclick = () => {
  modal.style.display = "none";
}

Sposób 2 - showModal() i close()

Wykorzystujemy wbudowane metody elementu dialog.
showModal() — otwiera okno jako modal
close() — zamyka okno

To jest modal otwarty metodą showModal().
const modal = document.getElementById("modal");
const openBtn = document.getElementById("openModal");
const closeBtn = document.getElementById("closeModal");

openBtn.onclick = () => {
  modal.showModal();
}

closeBtn.onclick = () => {
  modal.close();
}

Sposób 3 - Invoker Commands API

Nowoczesne API, które pozwala nam pisać w guziku polecenia.
Nowoczesny sposób otwierania i zamykania modala bez JavaScript.

Ten modal został otwarty bez JavaScript.
<button commandfor="modal" command="show-modal"> Otwórz modal </button>

<dialog id="modal">
  <button commandfor="modal"   command="close">   Zamknij   </button>
</dialog>

Sposób na zamknięcie, gdy klikniemy poza dialog

Do 1 i 2 sposobu możemy dopisać następującego JS:

modal.addEventListener("click", (e) => {
 if (e.target === modal) {
  modal.style.display = "none";
  //lub
  modal.close()
 }
});

Dzięki temu, jeśli klikniemy poza obszar modalu zostanie on zmaknięty

Modal wyłączy się jak klikniesz poza.

Dlaczego showModal() jest lepsze?

Automatycznie blokuje stronę.
Ma wbudowaną metodę zamknięcia: close()

Animacja 3D pojawiania się modalu.

/* Stylizacja modala */
dialog {
 opacity: 0;
 transform: perspective(2000px) rotateX(-90deg);
 transform-origin: top center;
 transition:
  opacity 0.5s ease-out,
  transform 0.5s cubic-bezier(0.34, 1.56, 0.64, 1),
  display 0.5s allow-discrete;
}

dialog[open] {
 opacity: 1;
 transform: perspective(2000px) rotateX(0deg);
}

@starting-style {
 dialog[open] {
  opacity: 0;
  transform: perspective(2000px) rotateX(-90deg);
 }
}

dialog::backdrop {
 background-color: rgba(0, 0, 0, 0);
 transition: background-color 0.5s ease, display 0.5s allow-discrete;
}

dialog[open]::backdrop {
 background-color: rgba(0, 0, 0, 0.6);
}

@starting-style {
 dialog[open]::backdrop {
  background-color: rgba(0, 0, 0, 0);
 }
}
