Na czym polega ćwiczenie?

Tworzymy mini wersję Wikipedii, w której po najechaniu na słowo pojawia się popover z definicją.

Pozycjonowanie dynamiczne

Używamy IntersectionObserver, aby sprawdzić, czy popover wychodzi poza ekran.

Jeśli element zbliża się do górnej krawędzi, dodajemy klasę up i przenosimy popup nad słowo.

Jeśli zbliża się do lewej krawędzi, dodajemy klasę left i przesuwamy go w drugą stronę.

JavaScript – logika działania

const options_y = {
    rootMargin: '-50% 0px 0px 0px',
    threshold: 0
}

const options_x = {
    rootMargin: '0px 0px 0px -50%',
    threshold: 0
}

const callback_y = (entries) => {
    entries.forEach(entry => {
        if (entry.isIntersecting) {
            entry.target.classList.add('up')
        } else {
            entry.target.classList.remove('up')
        }
    })
}

const callback_x = (entries) => {
    entries.forEach(entry => {
        if (entry.isIntersecting) {
            entry.target.classList.add('left')
        } else {
            entry.target.classList.remove('left')
        }
    })
}

document.querySelectorAll('.def').forEach(element => {
    const observer_x = new IntersectionObserver(callback_x, options_x)
    observer_x.observe(element)

    const observer_y = new IntersectionObserver(callback_y, options_y)
    observer_y.observe(element)
})

Tryb mobilny

Na mniejszych ekranach popover jest przypięty do dołu ekranu i zajmuje całą szerokość.

Dzięki temu jest czytelny i wygodny w obsłudze na telefonach.

Zmienia się też sposób aktywacji – zamiast hover używamy kliknięcia.
