Jak zbudować nowoczesną galerię zdjęć?

Ten dokument zawiera w pełni działający przykład zaawansowanej galerii siatkowej. Kliknij dowolne zdjęcie poniżej, aby przetestować działanie modala z płynną animacją 3D.

Podgląd Live Projektu

Jak to zostało zrobione? Krok po kroku.

Krok 1: Struktura HTML

Każdy element galerii to div z klasą .image, w którym znajduje się standardowy znacznik <img>. Dodatkowo na dole strony znajduje się nowoczesny tag <dialog> służący jako modal okna podglądu.

<div class="gallery">
    <div class="image"><img src="sciezka.jpg" alt="Opis"></div>
</div>

<dialog>
    <div class="dialog-menu"><button onclick="closeModal()">Zamknij</button></div>
    <div id="dialog-image-container"></div>
</dialog>

Krok 2: Elastyczna Siatka CSS Grid

Użycie grid-template-columns: repeat(auto-fill, minmax(min(150px, 100%), 1fr)); sprawia, że przeglądarka sama decyduje, ile kolumn zmieści się na ekranie. Kafelki automatycznie zmniejszają się i zwiększają, dopasowując się do telefonów i monitorów bez pisania Media Queries!

Krok 3: Efekt inteligentnego gradientu w tle (CSS Custom Properties)

Aby tło miało idealnie dopasowaną kolorystykę, wykorzystujemy to samo zdjęcie, ale mocno zamazane. Robimy to przy pomocy pseudoelementu ::before:

.gallery .image::before {
    content: "";
    position: absolute;
    inset: 0;
    /* Pobiera adres URL przypisany dynamicznie przez JavaScript */
    background: var(--bg-url) center/cover no-repeat;
    filter: blur(20px) saturate(150%);
    transform: scale(1.1); /* Ukrywa białe krawędzie powstałe przez blur */
    z-index: 1;
}
Dlaczego stosujemy ::before, a nie zwykły background na divie?
Gdybyśmy nałożyli filter: blur() bezpośrednio na kontener .image, rozmyłoby się wszystko wewnątrz niego – włącznie z właściwym, ostrym zdjęciem. Dzięki ::before rozmywamy tylko warstwę tła pod spodem, a właściwy obrazek zachowuje 100% ostrości.

Krok 4: Skrypt JavaScript dla Tła i Modala

JavaScript w tym projekcie wykonuje dwa zadania:

  1. Pętla przechodzi przez kafelki, wyciąga z nich adres `src` zdjęcia i zapisuje go jako zmienną CSS --bg-url. Dzięki temu tło w CSS wie, jaki obrazek rozmyć.
  2. Obsługuje otwieranie wbudowanego w HTML5 modala za pomocą natywnej funkcji showModal().
const gallery = document.querySelector('.gallery');
const images = gallery.querySelectorAll('.image');

// 1. Automatyczne ustawianie tła ze zmiennej CSS
images.forEach(imgContainer => {
    const imgSrc = imgContainer.querySelector('img').src;
    imgContainer.style.setProperty('--bg-url', `url(${imgSrc})`);
});

// 2. Obsługa Modala Dialog
const modal = document.querySelector('dialog');
images.forEach(image => {
    image.addEventListener('click', e => {
        const targetImg = e.currentTarget.querySelector('img');
        const clonedImg = targetImg.cloneNode(true);
        
        const container = document.getElementById('dialog-image-container');
        container.innerHTML = '';
        container.appendChild(clonedImg);
        
        modal.showModal(); // Natywne, bezpieczne otwarcie okna podglądu
    });
});

function closeModal() {
    modal.close();
}

Krok 5: Nowoczesna animacja 3D typu "Falldown"

Do animacji okna użyliśmy nowoczesnych reguł CSS: właściwości transition: ... display 0.4s allow-discrete; oraz reguły @starting-style. Pozwala to na płynne animowanie elementów, które zmieniają swój stan z ukrytego (display: none) do widocznego blokowo, co dawniej wymagało skomplikowanego kodu JS.
