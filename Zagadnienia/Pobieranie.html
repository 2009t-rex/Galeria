O co chodzi z pobieraniem zdjęć?

Każdy wie, że da się pobrać zdjęcia klikając na nie prawym myszki.
Robimy galerie, więc robimy to ładnie, czyli na dwa sposoby, wymagany 2 na zaliczenie.

Pierwszy sposób anchor z atrybutem dowload

Najprostszy sposób pobrania obrazka to użycie linku z atrybutem download:

<a href="zdjecie.format" download>Pobierz zdjęcie</a> Pobierz zdjęcie

Po kliknięciu przeglądarka pobierze plik zamiast go otwierać.

Drugi sposób za pomocą guzika

To samo, tylko tym razem w Javascript, button można ładniej stylizować niż znacznik <a>

Plusy:

Ten sposób pozwala na pobranie nawet jeśli źródło się zmieni. Przykład:

Przykład ze zmianą zdjęcia.

//Obrazek ma id obrazek, guzik id zapiszFile
const zapiszFile = document.getElementById("zapiszFile");
const obrazek = document.getElementById("obrazek");

zapiszFile.addEventListener("click", () => {
const save = document.createElement("a");
save.href = obrazek.src;

// Pobranie nazwy pliku z adresu obrazka
const nazwaPliku = obrazek.src.split("/").pop()
save.download = nazwaPliku;

document.body.appendChild(save);
save.click();
document.body.removeChild(save);
});
