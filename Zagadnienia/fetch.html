Co to fetch()?

fetch() - wbudowana funkcja w JS do wysyłania żądań HTTP do serwerów i pobierania danych.

Jest obecnie standardowym sposobem wykonywania zapytań sieciowych w JavaScript i zastąpił starsze rozwiązanie XMLHttpRequest

Ważniejsze zastosowania

  • Pobieranie danych z API - rzeczy typu pobieranie pogody, kursów walut lub wiadomości.
  • Ładowanie danych bez odświeżania strony.
  • Wysyłanie formularzy do serwera.
  • Logowanie i rejestracja użytkowników.
  • Pobieranie plików JSON.
  • Komunikacja z bazą danych za pośrednictwem backendu.
  • Pobieranie tekstu, obrazów i innych zasobów z serwera.

Przykład:

Tutaj jest wzięty z api o losowych żartach

// Wysyła żądanie HTTP do podanego adresu API
fetch("https://official-joke-api.appspot.com/random_joke")

// Po otrzymaniu odpowiedzi zamienia JSON na obiekt JavaScript
.then(response => response.json())

// Dane przypisujemy do zmiennej data i ją wyświetlamy w konsoli,
// żeby zobaczyć budowę obiektu
.then(data => {
console.log(data);
})
// Łapie błędy, chyba oczywiste...
.catch(error => {
console.error(error);
});
